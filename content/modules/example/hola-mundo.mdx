# ¡Hola Mundo! — Tu Primer Programa en Python

Bienvenido al mundo de la programación. En esta lección vas a escribir tu primer programa en Python.

## ¿Qué es `print()`?

La función `print()` muestra texto en la pantalla. Es una de las funciones más básicas y útiles de Python.

Ejecutá este código para ver cómo funciona:

<CodeEditor
  defaultValue='print("Hola Mundo")'
  height="200px"
  language="python"
/>

## Probalo Vos

Cambiá el texto entre comillas y ejecutalo de nuevo. Por ejemplo:

<CodeEditor
  defaultValue='print("Este es mi primer programa!")'
  height="200px"
  language="python"
/>

## Lo que aprendiste

- `print()` muestra texto en la consola
- El texto va entre comillas `" "`
- Python ejecuta el código línea por línea
